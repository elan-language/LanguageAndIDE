The Elan Reference Manual

Elan Reference Manual conists of the following sections:

  • The IDE Reference explains the Code Editor and other features of this IDE.
  • The Language Reference explains individual instructions, the language syntax, and compile rules.
  • The Library Reference provides detail on library types, and methods.
  • The Index lists all keywords, Types, methods, punctuation symbols, and topics, with links into the other sections.

Note that the Reference Manual button at the bottom of this tab gives direct access to each of these sections
