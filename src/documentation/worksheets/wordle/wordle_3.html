Step 1: introduction

Following-on from 'Wordle 2: writing an automated solver', in this worksheet your challenge is to write an analyser that can evaluate:

  1. Just how effective the automated solver is at solving puzzles. This will be done by getting it attempt to solve all 2,315 of the possible target words, and measuring the percentage that it solves within six attempts and, for those successful cases, the average number of attempts required.
  2. Whether ARISE is the best starting word or, if not, what is the best starting word.

and note the following:

  • Underneath all the imports you will see all the code you have written in the previous projects.
  • The new code currently has no main routine - but you will be adding one.
  • There are two ghosted tests at the bottom, each covering several test cases and referring to functions that do not yet exist, but you will be writing.
