Version history

The .zip of the latest version can be installed on your own webserver.

Elan adopts semantic versioning:

  • A third-level release (e.g. x.x.2) is either a bug-fix or a change to the IDE that involves no change to the language or library.
  • A second-level release (e.g. x.2.0) indicates a backwards-compatible extension of the language or library.
  • A first-level release (e.g. 2.0.0) would imply a breaking change to language or library. We aim not to make any such releases, unless there were a compelling case to do so - and in that case it would be accompanied by an upgrade utility to facilitate translating existing Elan code.
