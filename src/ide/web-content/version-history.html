Version history

The .zip of the latest version can be installed on your own webserver.

Elan adopts semantic versioning:

  • A third-level release (e.g. x.x.2) is either a bug-fix or a change to the IDE that involves no change to the language or library.
  • A second-level release (e.g. x.2.0) indicates a backwards-compatible extension of the language or library.
  • A first-level release (e.g. 2.0.0) would imply a breaking change to language or library. Well in advance of any such breaking change we will put in 'advisory' messages on any code that will be changing, and advise how best to anticipate the change. We will also continue to provide online access to the last version before any first level release, so that you are always guaranteed to be able to run your programs without modification.
